# lern-flask
lerning flask

## env setting 

```sh
virtualenv env
```
### linux

source env/bin/activate

### windows

env\Script\activate

## install 

```sh
pip install flask
```

## run flask

```sh
flask --app main run -p 3000 -h 0.0.0.0

flask --app main run -p 3000 -h 0.0.0.0 --debug

python -m flask --app main run -p 3000 -h 0.0.0.0
```
